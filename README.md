# monster-siren

仿 [塞壬唱片 - A WORLD FAMILIARLY UNKNOWN](https://monster-siren.hypergryph.com/) 项目

***预览：*** <https://monster-siren.YuePlus.ink/>

## 参与开发

获取本项目

```bash
git clone https://github.com/Yue-plus/monster-siren.git
cd monster-siren/
```

安装依赖

```bash
yarn
```

开启本地开发服务器

```bash
yarn start
```
